CSS

캐스 케이 딩 스타일 시트 ( CSS은 )는 인 스타일 시트 언어 묘화에 사용되는 프리젠 테이션 에 작성하는 문서의 마크 업 언어를 같은 HTML. CSS는 HTML 및 JavaScript 와 함께 World Wide Web 의 초석 기술입니다. CSS는 레이아웃 , 색상 및 글꼴을 포함하여 프레젠테이션과 콘텐츠를 분리 할 수 ​​있도록 설계되었습니다. 이러한 분리는 컨텐츠 접근성을 향상시키고 , 프리젠 테이션 특성의 사양에서 더 많은 유연성과 제어를 제공 하며, 구조적 컨텐츠의 복잡성과 반복을 줄여주는 별도의 .css 파일에 관련 CSS를 지정하여 여러 웹 페이지 에서 서식을 공유 할 수 있도록 합니다. .css 파일을 캐시 할 수 있도록 설정하여 파일을 공유하는 페이지와 해당 형식 간의 페이지로드 속도를 향상시킵니다.
